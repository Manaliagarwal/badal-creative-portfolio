# Images Folder

Place your portfolio images here.

## Recommended Structure

```
assets/images/
├── hero.jpg              # Hero section image (1200x600px)
├── profile.jpg           # Your profile photo (600x600px)
├── about-bg.jpg          # About page background (optional)
├── projects/
│   ├── project-1.jpg     # Project thumbnails (800x500px)
│   ├── project-2.jpg
│   ├── project-3.jpg
│   └── ...
└── testimonials/
    ├── client-1.jpg      # Client photos (150x150px)
    ├── client-2.jpg
    └── ...
```

## Image Guidelines

### Dimensions
- **Hero Images**: 1200x600px or 1920x1080px
- **Profile Photo**: 600x600px (square, centered)
- **Project Thumbnails**: 800x500px (16:10 ratio)
- **Client Avatars**: 150x150px (square)

### File Sizes
- Target: Under 200KB per image
- Maximum: 500KB per image
- Use compression tools:
  - [TinyPNG](https://tinypng.com/)
  - [Squoosh](https://squoosh.app/)
  - [ImageOptim](https://imageoptim.com/)

### Formats
- **Preferred**: JPG for photos, PNG for graphics
- **Modern**: WebP for better compression
- **Avoid**: Unoptimized RAW files

### Naming Convention
- Use lowercase
- Use hyphens (not spaces)
- Be descriptive
- Examples:
  - `hero-workspace.jpg`
  - `project-brand-video.jpg`
  - `profile-photo-2024.jpg`

## Quick Tips

1. **Always optimize** images before uploading
2. **Use descriptive** alt text in HTML
3. **Keep backups** of original high-res files elsewhere
4. **Update paths** in HTML after adding images
5. **Test loading** speed after adding images
